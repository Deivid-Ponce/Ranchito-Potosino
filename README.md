El Ranchito Potosino – Sitio web de catálogo y pedidos
Landing page para una tienda de frutas y verduras a domicilio, con sección de productos destacados, navegación por categorías (Boxes), banner promocional, sección “Nosotros” y formulario de contacto. Incluye integración visual con Bootstrap 5, estilos personalizados en SCSS/CSS, y lógica en JavaScript para calcular totales por tipo de caja (Básica, Plus, Premium y Lunch).
Tecnologías: HTML5, CSS3/SCSS, JavaScript, Bootstrap 5.
